# Occupational Builder

A stubbed Vite + React + TypeScript project for the Occupational Builder v1 prototype.

## Getting started
1. Install dependencies with `npm install`.
2. Start local development with `npm run dev`.
3. Create a production build with `npm run build`.
4. Preview the build locally with `npm run preview`.

The Vite configuration uses `base: "./"` to support GitHub Pages deployment. Units throughout the app are expressed in millimeters (mm) to align with accessibility construction requirements.
